Prototyping a Drum Sequencer for Visually Impaired Users  <br>
Edited: https://nyumusedlab.github.io/Accessible-Groove-Pizza/

### Default
Number 1 - Switch drum sets
			Q - drum set 1 
			A - drum set 2
			W - drum set 3
Number 2 - inner layer
Number 3 - middle layer
